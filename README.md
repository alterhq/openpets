# OpenPets

OpenPets provides a desktop pet that AI assistants can use as a visible task-state channel for progress, completion, failures, and review requests.

## AI Assistant Instructions

If you use OpenPets through an AI coding assistant, add the recommended instructions so the assistant uses the desktop pet consistently instead of treating notifications as optional.

See [docs/ai-assistants](./docs/ai-assistants/) for setup guidance covering OpenCode, Claude Code, Cursor, Pi CLI, and generic MCP clients.
